# Public Launch

TerraGrid Atlas is public and open to contributors.

## Main message

TerraGrid Atlas is a Next.js and TypeScript web app for exploring map-based data, dashboards, filters, comparison views, and sample records.

The project is useful for people who want to practice:

- Next.js
- TypeScript
- Tailwind CSS
- MapLibre
- Recharts
- data display
- dashboard design
- responsive UI

## How people can follow the project

People can support the project by:

- starring the repository
- watching the repository for updates
- opening issues for improvements
- picking small roadmap tasks
- submitting focused pull requests

## Good starter areas

- documentation improvements
- screenshot updates
- small-screen layout polish
- filter tests
- empty-state polish
- sample record notes
- setup notes

## Safety rules

Contributors should not commit secrets, `.env` files, private datasets, private logs, screenshots with private information, tokens, or deployment secrets.
