# Roadmap

TerraGrid Atlas is a portfolio web app with maps, dashboards, filters, and sample data.

## Near term

- Deploy a public demo.
- Add the live URL to the README.
- Improve small-screen layout.
- Add clearer notes for sample records.
- Improve empty states.
- Add more contributor docs.

## Medium term

- Add tests for filters and data helpers.
- Improve comparison charts.
- Improve loading states.
- Improve map controls.
- Add more example records.

## Later

- Add saved views.
- Add richer reports.
- Add stronger source notes.
- Add more review commands.

## Principles

- Keep sample data clearly labeled.
- Keep the project easy to run locally.
- Keep verification commands documented.
- Keep the code easy to inspect.
