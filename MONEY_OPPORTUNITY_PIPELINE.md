## LinkedIn Post Draft

I just finished polishing TerraGrid Atlas, a map-first infrastructure intelligence project I built to explore how global power systems are distributed and connected.

The app combines a dark command-center world map, power plant and substation layers, transmission corridors, data center markers, country dashboards, comparison tools, Recharts analytics, and a transparent methodology page.

The current version uses clearly labeled sample/demo data so it can run without paid services, but the architecture is typed around provider adapters for future public sources like OpenStreetMap, power plant datasets, regional load feeds, and country energy statistics.

This was a chance to practice Next.js, TypeScript, Tailwind, MapLibre, data modeling, dashboard UX, and honest product documentation in one project.

## Handshake Showcase Submission

Project title: TerraGrid Atlas

Short description:
A global interactive atlas for exploring power plants, substations, transmission systems, data centers, and grid intelligence.

Full description:
I built TerraGrid Atlas to make global energy infrastructure easier to explore and understand. The app brings together a dark geospatial infrastructure map, power plants, substations, transmission corridors, data centers, grid regions, alerts, country dashboards, comparison tools, generation mix charts, and a transparent methodology page into one research-friendly interface.

The current version uses clearly labeled sample/demo data and generated visual density points so the app can run without paid services or a required database. I do not present it as fully live global infrastructure coverage. The code is structured around typed data models and provider adapters so real public datasets such as OpenStreetMap, public power plant datasets, regional load feeds, and country energy statistics can be connected later.

What makes it original is that it is not a chatbot. It combines geospatial mapping, infrastructure intelligence, dashboard analytics, and honest sample-data architecture into a visual research tool that students, researchers, and citizens can use to understand energy systems.
