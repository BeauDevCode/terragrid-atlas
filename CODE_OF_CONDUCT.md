# Code of Conduct

This project should be a respectful and useful place to collaborate.

## Expected behavior

- Be respectful.
- Keep feedback specific and helpful.
- Stay focused on the project.
- Assume good intent when possible.
- Help new contributors understand the codebase.

## Unacceptable behavior

- Harassment or personal attacks.
- Sharing private information.
- Posting spam.
- Adding harmful or misleading content.
- Submitting private data, secrets, or credentials.

## Reporting

If there is a problem, contact the maintainer privately.

## Maintainer responsibility

The maintainer may edit, close, or remove issues, pull requests, or comments that make the project unsafe or unproductive.
